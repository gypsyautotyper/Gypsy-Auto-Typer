Frequently Asked Questions

What operating systems are supported?

Currently, Gypsy AutoTyper supports Windows only. macOS and Linux versions are coming soon.

Is Gypsy AutoTyper safe to install?

Yes—our installer is lightweight, contains no malware or tracking software, and runs completely offline once installed.

How do I customize typing speed and error rate?

Use the sliders on the interface to set your preferred characters-per-minute (CPM) and typo frequency, then click “Start.”

Can I use Gypsy AutoTyper offline?

Absolutely—once installed, you don’t need an internet connection to run the application.
